Quick Start

Setup in 3 Steps

  • 1. Install dependencies: pip install -r requirements.txt
  • 2. Run the app: python main.py
  • 3. Control from tray: Right-click the tray icon to start/stop capture

Key Features

  • ⏱️ Automatic screenshot capture at configurable intervals
  • 🖼️ Timestamped PNG files organized by capture session
  • ⚙️ Preset (5s, 10s, 30s, 60s) and custom interval support
  • 📊 Screenshot count display in tray menu
  • 📂 Direct access to screenshots folder from tray
  • 🔍 Comprehensive logging for diagnostics
  • 🎨 Modern, user-friendly custom interval dialog

📦 System Requirements

  • Python 3.7+
  • Windows OS (primary target)
  • pillow (image processing)
  • pystray (system tray integration)
